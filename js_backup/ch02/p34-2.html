자바스크립트를 작성하는 규칙

1. 대소문자를 구분해서 작성한다.
2. 들여쓰기를 해서 작성한다.
3. 행의 끝은 세미콜론으로 마무리한다.
4. 주석을 사용하여 설명을 작성한다.
5. 식별자는 정해진 규칙을 지켜 작성한다.
ex) 변수, 함수, 속성, 클래스 ...
6. 예약어는 식별자로 사용할 수 없다.

변수

변수 : 값(데이터)을 저장하는 공간

변수를 선언하는 규칙

1. 이름을 의미있게 짓는다.(권장)
2. 변수 이름은 카멜 케이스(Camel Case) (권장)
3. 변수의 이름은 대소문자를 구분한다.
4. 변수 이름의 첫 글자는 영어 대소문자, _(언더바), $(달러)를 사용, 두 번째 글자부터는 숫자도 사용할 수 있다.
- $(달러)는 사용하지 않기를 권장 -> 시스템 변수 앞머리에 사용되므로(구분)
5. 예약어는 변수이름으로 사용 불가
