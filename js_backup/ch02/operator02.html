연산자(operator)

3. 비교연산자 : >, >=, <, <=, ==(equal), !=(not equal), ===, !==
- ==, != : 값만 비교해서 true / false를 출력
- ===, !== : 값과 데이터타입을 모두 비교해서 true / false를 출력
- 우선순위 : (>, >=, <. <=) 연산자가 (==, !==)보다 우선
- 우선순위 : (==, !=) 연산자가 (===, !==)보다 우선
- (>, >=, <. <=) > (==, !=) > (===, !==)
4. 논리연산자 : &&(논리 And), ||(논리 Or), !(논리 Not, 논리 부정)
- 논리연산자의 결과는 논리값(true, false 둘 중의 하나)
- 비교연산자가 논리연산자보다 우선(비교 > 논리)
- 우선순위 : 논리AND가 논리OR보다 우선임
- 우선순위 : ! > && > ||
5. 대입연산자 : =
복합대입연산자(할당연산자, assignment) : +=, -=, *=, /=, %=
